# Resources Folder Outline

Top-level sketch of materials currently in `resources/` and their expected
relevance to this project.

## High relevance (current roadmap)

- `RL text.pdf`
  - Comprehensive reinforcement learning textbook reference.
  - Useful for algorithm descriptions, notation, and doc background sections.
- `DeepQNetworks_Nature.pdf`
  - Canonical DQN paper.
  - Useful when documenting value-based deep RL baseline rationale.
- `dueling DQN.pdf`
  - Dueling architecture extension to DQN.
  - Useful for future RL model variants page/doc section.
- `prioity replay.pdf`
  - Prioritized experience replay.
  - Useful for future data-sampling strategy docs.
- `RL_cheatsheet.pdf`
  - Quick reference for RL concepts and equations.
  - Useful for concise glossary-style documentation.
- `RL walk learning.pdf`
  - Tutorial-style RL material.
  - Useful for onboarding-oriented explanatory docs.

## Medium relevance (later-phase advanced work)

- `DeepNash.pdf`
  - Advanced self-play/game-theoretic system paper.
  - Useful for long-term strategy self-play section, less critical for v1.

## Low relevance to current RPS roadmap

- `dcgan_paper.pdf`
  - GAN/image-generation focused.
  - Not directly relevant unless future work includes generative modeling.
